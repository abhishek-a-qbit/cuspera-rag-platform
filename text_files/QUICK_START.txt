# Quick Start Guide

## ðŸš€ Running the Cuspera RAG Application

### Prerequisites
1. **Python 3.10+** installed
2. **GOOGLE_API_KEY** from Google AI (Gemini API)

### Step 1: Setup Environment File

```bash
# Copy the example config
Copy-Item config/.env.example .env

# Open .env and add your GOOGLE_API_KEY
# Example:
# GOOGLE_API_KEY=your_actual_api_key_here
```

### Step 2: Install Dependencies

```bash
pip install -r config/requirements.txt
```

### Option A: Run Just the Streamlit App (Simple)

```bash
# From the root directory
streamlit run app/streamlit_app.py
```

The app will open at: `http://localhost:8501`

**Note**: This runs the frontend only. It connects to API endpoints (requires API running separately if you want full functionality).

### Option B: Run Complete System (API + Frontend)

#### Terminal 1 - Start the Backend API:
```bash
cd src
python api_backend.py
```

API will run at: `http://localhost:8000`

#### Terminal 2 - Start the Frontend:
```bash
streamlit run app/streamlit_app.py
```

Frontend will run at: `http://localhost:8501`

### Option C: Use the PowerShell Launcher (Windows)

```bash
.\RUN_APP.ps1
```

This will:
1. Check for .env file
2. Install dependencies if needed
3. Start the Streamlit app

## ðŸŽ¯ What Each Component Does

### Streamlit App (`app/streamlit_app.py`)
- **Chat**: Ask questions about 6sense platform
- **Analytics**: See platform insights
- **Reports**: Generate feature reports  
- **Status**: Monitor system health

### Backend API (`src/api_backend.py`)
- REST API on port 8000
- 7+ endpoints for retrieval, generation, analytics
- RAG pipeline orchestration

### Vector Store (`src/vector_store.py`)
- Manages embeddings with ChromaDB
- Hybrid search (semantic + keyword)
- Document indexing and retrieval

### Hybrid Search (`src/hybrid_search.py`)
- Combines semantic understanding with keyword matching
- Configurable weights (default 60/40)
- Better accuracy than semantic alone
