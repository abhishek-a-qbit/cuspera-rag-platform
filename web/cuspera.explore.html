Module Capabilities

The system is composed of four distinct intelligent agents working in concert.

A

The Dispatcher

Routes queries to semantic namespaces (e.g., "Lead Gen" → CRM Database).

B

The Analyst

Maps user problems found in uploaded PDFs to feature solutions in software docs.

C

The Matchmaker

Embedding similarity search followed by LLM-based re-ranking for nuance.

D

The Visualizer

Python-based agents generating on-the-fly charts for financial/feature comparisons.

Trust is Metric-Driven

We don't guess if the system works; we score it. Using the RAGAS Framework, we measure every interaction against three critical pillars: Faithfulness, Context Precision, and Answer Relevance.

Metric 1 Faithfulness Did the bot hallucinate features?
Metric 2 Context Precision Did it cite the correct pricing PDF?

Performance vs. Standard RAG

Human-in-the-Loop (HITL)

Experts review recommendations. If the AI suggests a tool outside the budget, the expert corrects it. This creates a "Golden Dataset" that re-trains the model, creating a compound interest effect on accuracy.
